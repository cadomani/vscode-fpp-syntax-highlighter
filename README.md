# fpp README

Vim-style highlighting within VSCode.

## Features

Supports:
  Keywords
  Types
  Comments
  Annotations
  Identifiers
  Strings

## Known Issues

Numeric / Scientific values not yet highlighted

## Release Notes

Initial release with basic syntax highlighting
